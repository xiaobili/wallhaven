# Plan 01: LoadingOverlay Animation Performance Optimization

<objective>
优化 LoadingOverlay 组件动画性能：移除 backdrop-filter blur，统一使用 animations.css，添加 reduced-motion 支持。
</objective>

<tasks>

## Task 1: Add fade transition classes to animations.css

<read_first>
- src/static/css/animations.css
</read_first>

<action>
在 `src/static/css/animations.css` 文件末尾添加以下内容：

```css
/* ===== Fade Transition (Vue Transition) ===== */

/**
 * Fade Transition Classes
 * Used by LoadingOverlay.vue and other components
 */
.fade-enter-active,
.fade-leave-active {
  animation: fade 0.2s ease-out both;
  will-change: opacity;
}

.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}

/* Reduced motion support for fade */
@media (prefers-reduced-motion: reduce) {
  .fade-enter-active,
  .fade-leave-active {
    animation: fade 0.15s ease-out both;
  }
}
```

注意：`@keyframes fade` 已存在于文件中（第45-52行），无需重复添加。
</action>

- `grep -n "will-change: opacity" src/static/css/animations.css` 返回匹配行
- 文件包含 `@media (prefers-reduced-motion: reduce)` 针对 fade 的处理
</acceptance_criteria>

## Task 2: Optimize LoadingOverlay.vue animation

<read_first>
- src/components/LoadingOverlay.vue
- src/static/css/animations.css
</read_first>

<action>
修改 `src/components/LoadingOverlay.vue`：

1. **移除 backdrop-filter:**
   - 删除 `.loading-overlay` 中的 `backdrop-filter: blur(2px);` 行

2. **添加 animations.css 导入:**
   - 在 `<style scoped>` 标签后添加导入语句：
   ```css
   @import url("@/static/css/animations.css");
   ```

3. **移除组件内的 fade 动画定义:**
   - 删除以下 CSS 块（约第60-69行）：
   ```css
   /* 淡入淡出动画 */
   .fade-enter-active,
   .fade-leave-active {
     transition: opacity 0.3s ease;
   }

   .fade-enter-from,
   .fade-leave-to {
     opacity: 0;
   }
   ```
   这些定义现在由 animations.css 提供。

4. **添加 reduced-motion 支持:**
   在 `</style>` 标签前添加：
   ```css
   /* Reduced motion: disable fa-spin animation */
   @media (prefers-reduced-motion: reduce) {
     .loading-icon.fa-spin {
       animation: none;
     }
   }
   ```
</action>

- `grep "backdrop-filter" src/components/LoadingOverlay.vue` 返回空（已移除）
- `grep "@import url" src/components/LoadingOverlay.vue` 返回 animations.css 导入
- `grep "transition: opacity 0.3s" src/components/LoadingOverlay.vue` 返回空（已移除）
- `grep "@media (prefers-reduced-motion: reduce)" src/components/LoadingOverlay.vue` 返回匹配行
- `grep "animation: none" src/components/LoadingOverlay.vue` 返回匹配行
</acceptance_criteria>
