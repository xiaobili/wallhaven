创建 WallpaperService，实现壁纸业务逻辑。

核心职责：
- 壁纸搜索 API 调用（使用 apiClient）
- 响应缓存（迁移自 wallpaperApi.ts）
- 查询参数持久化（使用 wallpaperRepository）
- API Key 内部注入（从 settingsRepository 获取）
</objective>

    <action>
      创建文件 `src/services/wallpaper.service.ts`，实现 WallpaperService 类：

      1. 添加正确的导入语句：
         ```typescript
         import type { IpcResponse } from '@/shared/types/ipc'
         import type { GetParams, CustomParams, WallpaperItem, WallpaperMeta } from '@/types'
         import { apiClient } from '@/clients'
         import { settingsRepository, wallpaperRepository } from '@/repositories'
         ```

      2. 定义导出接口 `WallpaperSearchResult`：
         ```typescript
         export interface WallpaperSearchResult {
           data: WallpaperItem[]
           meta: WallpaperMeta
         }
         ```

      3. 创建类 `WallpaperServiceImpl`，包含：
         - 私有缓存 Map：`private cache = new Map<string, { data: unknown; timestamp: number }>()`
         - 缓存常量：`CACHE_TTL = 5 * 60 * 1000`（5分钟）
         - `MAX_CACHE_SIZE = 50`

      4. 实现私有方法：
         - `generateCacheKey(url: string, params?: unknown): string` - 返回 `${url}:${JSON.stringify(params || {})}`
         - `getFromCache<T>(key: string): T | null` - 检查缓存是否存在且未过期
         - `setCache(key: string, data: unknown): void` - 设置缓存，超过限制时删除最旧条目
         - `getApiKey(): Promise<string | undefined>` - 调用 `settingsRepository.get()` 获取 apiKey

      5. 实现公开方法：
         - `async search(params: GetParams | null): Promise<IpcResponse<WallpaperSearchResult>>`
           * 过滤空值参数
           * 检查缓存
           * 调用 `apiClient.get<WallpaperSearchResult>('/search', finalParams, apiKey)`
           * 成功时缓存结果
         - `async getDetail(id: string): Promise<IpcResponse<WallpaperItem>>`
           * 检查缓存
           * 调用 `apiClient.get<WallpaperItem>(\`/w/${id}\`, undefined, apiKey)`
         - `async saveQueryParams(params: CustomParams): Promise<IpcResponse<void>>`
           * 确保 selector 为 0
           * 调用 `wallpaperRepository.setQueryParams({ ...params, selector: 0 })`
         - `async loadQueryParams(): Promise<IpcResponse<CustomParams | null>>`
           * 调用 `wallpaperRepository.getQueryParams()`
         - `clearCache(): void` - 清空缓存 Map

      6. 导出单例：
         ```typescript
         export const wallpaperService = new WallpaperServiceImpl()
         ```
