      创建文件 `src/services/download.service.ts`，实现 DownloadService 类：

      1. 添加正确的导入语句：
         ```typescript
         import type { IpcResponse } from '@/shared/types/ipc'
         import type { FinishedDownloadItem } from '@/types'
         import { electronClient } from '@/clients'
         import { settingsRepository, downloadRepository } from '@/repositories'
         ```

      2. 定义导出类型：
         ```typescript
         export interface DownloadProgressData {
           taskId: string
           progress: number
           offset: number
           speed: number
           state: 'downloading' | 'paused' | 'waiting' | 'completed'
           filePath?: string
           error?: string
         }
         
         export type ProgressCallback = (data: DownloadProgressData) => void
         ```

      3. 创建类 `DownloadServiceImpl`，包含：
         - `private progressCallbacks = new Set<ProgressCallback>()`
         - `private isListenerRegistered = false`

      4. 实现构造函数（注册 Electron 进度监听）：
         ```typescript
         constructor() {
           // 检查 electronClient 是否可用后再注册监听
           if (typeof window !== 'undefined' && window.electronAPI) {
             this.registerProgressListener()
           }
         }
         ```

      5. 实现私有方法：
         - `registerProgressListener(): void`
           * 检查 `isListenerRegistered`，避免重复注册
           * 调用 `electronClient.onDownloadProgress((data) => { ... })`
           * 遍历 `progressCallbacks` 调用每个回调
           * 使用 try-catch 保护每个回调

      6. 实现公开方法：
         - `onProgress(callback: ProgressCallback): () => void`
           * 添加回调到 Set
           * 返回取消订阅函数：`() => this.progressCallbacks.delete(callback)`
         
         - `async getDownloadPath(): Promise<IpcResponse<string>>`
           * 调用 `settingsRepository.get()` 获取 downloadPath
           * 如果存在，返回 `{ success: true, data: downloadPath }`
           * 如果不存在，调用 `electronClient.selectFolder()` 提示用户选择
           * 选择成功后保存到设置并返回
           * 选择失败返回错误响应
         
         - `async startDownload(taskId: string, url: string, filename: string): Promise<IpcResponse<string>>`
           * 调用 `getDownloadPath()` 获取下载目录
           * 失败时返回错误响应
           * 成功时调用 `electronClient.startDownloadTask({ taskId, url, filename, saveDir })`
         
         - `async saveFinishedRecord(item: FinishedDownloadItem): Promise<IpcResponse<void>>`
           * 调用 `downloadRepository.add(item)`
         
         - `async getFinishedRecords(): Promise<IpcResponse<FinishedDownloadItem[]>>`
           * 调用 `downloadRepository.get()`
         
         - `async removeFinishedRecord(id: string): Promise<IpcResponse<void>>`
           * 调用 `downloadRepository.remove(id)`
         
         - `async clearFinishedRecords(): Promise<IpcResponse<void>>`
           * 调用 `downloadRepository.clear()`

      7. 导出单例：
         ```typescript
         export const downloadService = new DownloadServiceImpl()
         ```
