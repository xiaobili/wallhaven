
```bash
grep -rn "console.error" electron/main/ipc/handlers/*.ts
```
Should return no results (all use logHandler).

4. **Dynamic import check for store:**
```bash
grep -n "import('../../index')" electron/main/ipc/handlers/store.handler.ts
```
Should return at least one match.
</action>

- 'select-folder', 'read-directory', 'delete-file', 'open-folder'
- 'download-wallpaper', 'start-download-task', 'download-progress'
- 'set-wallpaper', 'save-settings', 'load-settings'

- select-folder: returns `string | null`
- read-directory: returns `{ error: string | null, files: LocalFile[] }`
- delete-file: returns `{ success: boolean, error: string | null }`

- All other handlers return their documented response types

Verify that the ElectronAPI interface in preload/index.ts has not changed its method signatures.
