## Goal

创建存储键常量文件，统一管理 electron-store 的键名，避免字符串拼写错误。

## Context

- 现有存储键名已固定：`appSettings`、`downloadFinishedList`、`wallpaperQueryParams`
- 键名在多个文件中硬编码，存在拼写风险
- 阶段 2 不改变键名，仅提取常量

## Tasks

### Task 1.1: 创建 constants.ts 文件

<read_first>
- src/stores/modules/wallpaper/settings-storage.ts (确认 appSettings 键名)
- src/stores/modules/download/index.ts (确认 downloadFinishedList 键名)
- src/stores/modules/wallpaper/storage.ts (确认 wallpaperQueryParams 键名)
</read_first>

<acceptance_criteria>
- 文件存在：src/clients/constants.ts
- 包含 STORAGE_KEYS 常量对象
- STORAGE_KEYS.APP_SETTINGS 值为 'appSettings'
- STORAGE_KEYS.DOWNLOAD_FINISHED_LIST 值为 'downloadFinishedList'
- STORAGE_KEYS.WALLPAPER_QUERY_PARAMS 值为 'wallpaperQueryParams'
- 使用 `as const` 确保类型安全
- TypeScript 编译无错误
</acceptance_criteria>

<action>
1. 创建目录 `src/clients/`（如果不存在）
2. 创建文件 `src/clients/constants.ts`
3. 添加以下内容：

```typescript
/**
 * 存储键常量
 * 统一管理 electron-store 的键名
 */

export const STORAGE_KEYS = {
  /** 应用设置存储键 */
  APP_SETTINGS: 'appSettings',
  /** 已完成下载列表存储键 */
  DOWNLOAD_FINISHED_LIST: 'downloadFinishedList',
  /** 壁纸查询参数存储键 */
  WALLPAPER_QUERY_PARAMS: 'wallpaperQueryParams',
} as const

export type StorageKey = (typeof STORAGE_KEYS)[keyof typeof STORAGE_KEYS]
```

4. 运行 `npx tsc --noEmit` 验证编译通过
</action>
