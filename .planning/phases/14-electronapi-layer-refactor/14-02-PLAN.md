```typescript
 * 在系统文件管理器中打开文件夹

  return settingsRepository.openFolder(folderPath)
}
```

2. **readDirectory 方法：**
```typescript

 * 读取目录内容
 * @param dirPath - 目录路径
 * @returns 返回目录中的文件列表

  return settingsRepository.readDirectory(dirPath)
}
```

3. **deleteFile 方法：**
```typescript
/**
 * 删除文件
 * @param filePath - 文件路径

}
```

需要：
1. 导入 LocalFile 类型（从 @/shared/types/ipc）
