<type>code</type>
<files>

</files>

- `src/composables/settings/useSettings.ts` (扩展的 composable)
</read_first>
<action>
修改 `src/views/LocalWallpaper.vue`：

1. **更新 import 语句**，添加新的 composables：
```typescript
import { useSettings, useAlert, useLocalFiles, useWallpaperSetter } from '@/composables'
```

2. **在 setup 中使用新的 composables**：
```typescript
const { settings } = useSettings()
const { alert, showSuccess, showError, hideAlert } = useAlert()
const { readDirectory, openFolder, deleteFile } = useLocalFiles()
const { setWallpaper } = useWallpaperSetter()
```

3. **替换直接调用**：
   - Line 68: `window.electronAPI.readDirectory(downloadPath.value)` → `await readDirectory(downloadPath.value)`
   - Line 100: `window.electronAPI.openFolder(downloadPath.value)` → `await openFolder(downloadPath.value)`
   - Line 146: `window.electronAPI.setWallpaper(imagePath)` → `await setWallpaper(imagePath)`
   - Line 166: `window.electronAPI.deleteFile(wallpaper.path)` → `await deleteFile(wallpaper.path)`

4. **调整返回值处理**：
   - readDirectory 返回 `IpcResponse<LocalFile[]>`，需要检查 `result.success` 和 `result.data`
   - openFolder 返回 `IpcResponse<void>`，需要检查 `result.success`
   - setWallpaper 返回 `boolean`（已在 composable 中处理提示）
   - deleteFile 返回 `IpcResponse<void>`，需要检查 `result.success`

注意：setWallpaper 返回 boolean 而非 IpcResponse，因为它已在 composable 中处理了成功/错误提示。
