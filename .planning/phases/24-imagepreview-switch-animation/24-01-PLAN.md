# Plan 24-01: Add Slide-Out Animation Classes to animate.css

<plan id="24-01">
<objective>Add slide-out-left and slide-out-right animation classes to animate.css, plus Vue transition wrapper classes for enter/leave animations</objective>

<read_first>
- src/static/css/animate.css (current animation definitions)
- .planning/phases/24-imagepreview-switch-animation/24-CONTEXT.md (D-04, D-05 decisions)

<truths>
- Animation duration MUST be 0.5s
- Easing MUST be cubic-bezier(0.25, 0.46, 0.45, 0.94)
- MUST include both -webkit- and standard keyframes for browser compatibility
- translateX values MUST match existing slide-in animations (1000px magnitude)
- slide-out-left: translateX 0 -> -1000px (exits to left)
- slide-out-right: translateX 0 -> 1000px (exits to right)
- Transition wrapper classes MUST include position: absolute for leave-active
</truths>

<must_haves>
- @keyframes slide-out-left definition with opacity: 1 -> 0
- @keyframes slide-out-right definition with opacity: 1 -> 0
- .slide-out-left class applying the animation
- .slide-out-right class applying the animation
- .slide-left-enter-active class using slide-in-left
- .slide-left-leave-active class using slide-out-right with position: absolute
- .slide-right-enter-active class using slide-in-right
- .slide-right-leave-active class using slide-out-left with position: absolute
</must_haves>

Append the following CSS to src/static/css/animate.css after the existing .slide-in-right class definition (around line 123):

```css
/* ----------------------------------------------
 * Slide-out animations for ImagePreview navigation
 * Phase 24: ImagePreview Switch Animation
 * ---------------------------------------------- */

/**
 * ----------------------------------------
 * animation slide-out-left (向左滑出)
 * ----------------------------------------
 */
@-webkit-keyframes slide-out-left {
  0% {
    -webkit-transform: translateX(0);
    transform: translateX(0);
    opacity: 1;
  }
  100% {
    -webkit-transform: translateX(-1000px);
    transform: translateX(-1000px);
    opacity: 0;
  }
}
@keyframes slide-out-left {
  0% {
    -webkit-transform: translateX(0);
    transform: translateX(0);
    opacity: 1;
  }
  100% {
    -webkit-transform: translateX(-1000px);
    transform: translateX(-1000px);
    opacity: 0;
  }
}
.slide-out-left {
  -webkit-animation: slide-out-left 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  animation: slide-out-left 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
}

/**
 * ----------------------------------------
 * animation slide-out-right (向右滑出)
 * ----------------------------------------
 */
@-webkit-keyframes slide-out-right {
  0% {
    -webkit-transform: translateX(0);
    transform: translateX(0);
    opacity: 1;
  }
  100% {
    -webkit-transform: translateX(1000px);
    transform: translateX(1000px);
    opacity: 0;
  }
}
@keyframes slide-out-right {
  0% {
    -webkit-transform: translateX(0);
    transform: translateX(0);
    opacity: 1;
  }
  100% {
    -webkit-transform: translateX(1000px);
    transform: translateX(1000px);
    opacity: 0;
  }
}
.slide-out-right {
  -webkit-animation: slide-out-right 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  animation: slide-out-right 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
}

/**
 * ----------------------------------------
 * Vue Transition wrapper classes
 * slide-left: prev navigation (new image from left, old image exits right)
 * slide-right: next navigation (new image from right, old image exits left)
 * ----------------------------------------
 */
.slide-left-enter-active {
  -webkit-animation: slide-in-left 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  animation: slide-in-left 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
}
.slide-left-leave-active {
  -webkit-animation: slide-out-right 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  animation: slide-out-right 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  position: absolute;
}

.slide-right-enter-active {
  -webkit-animation: slide-in-right 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  animation: slide-in-right 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
}
.slide-right-leave-active {
  -webkit-animation: slide-out-left 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  animation: slide-out-left 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) both;
  position: absolute;
}
```
