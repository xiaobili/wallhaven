# Plan 3: Add Favorite Button to ImagePreview

## Objective

Add favorite button to the ImagePreview sidebar for adding/removing wallpapers from collections during preview mode.

## Tasks

### Task 3.1: Add favorite button to ImagePreview sidebar

<read_first>
- `src/components/ImagePreview.vue` — Current sidebar structure with sidebar-fixed_box buttons
- `src/components/favorites/CollectionDropdown.vue` — Dropdown component for collection selection
- `src/composables/favorites/useFavorites.ts` — isFavorite function interface
</read_first>

<action>
Modify `src/components/ImagePreview.vue`:

1. **Add props for favorites:**
```typescript
interface Props {
  showing: boolean;
  imgInfo: WallpaperItem | null;
  isLocal: boolean;
  wallpaperList?: WallpaperItem[];
  currentIndex?: number;
  favoriteIds?: Set<string>;  // NEW: Set of favorited wallpaper IDs
}
```

2. **Add emits for favorite operations:**
```typescript
const emit = defineEmits<{
  close: [value: boolean];
  'set-bg': [item: WallpaperItem];
  'download-img': [item: WallpaperItem];
  navigate: [direction: 'prev' | 'next'];
  'toggle-favorite': [item: WallpaperItem, event: MouseEvent];  // NEW
}>();
```

3. **Add favorite button in sidebar (between "设为壁纸" and "下载"):**
```vue
<div class="sidebar-fixed_box favorite-btn"
  :class="{ 'is-favorite': isFavorite }"
  :title="isFavorite ? '已收藏' : '添加到收藏'"
  @click="handleFavoriteClick"
>
  <div class="icon-wrap">
    <i :class="isFavorite ? 'fas fa-heart' : 'far fa-heart'" />
  </div>
</div>
```

4. **Add isFavorite computed and handleFavoriteClick:**
```typescript
const isFavorite = computed(() => {
  if (!props.imgInfo) return false
  return props.favoriteIds?.has(props.imgInfo.id) || false
})

const handleFavoriteClick = (event: MouseEvent) => {
  if (!props.imgInfo) return
  emit('toggle-favorite', props.imgInfo, event)
}
```
</action>

- ImagePreview.vue has 'toggle-favorite' emit in defineEmits
- Template contains .favorite-btn element inside details-sidebar-fixed-box
- isFavorite computed property exists checking favoriteIds
- handleFavoriteClick function emits 'toggle-favorite' with imgInfo and event
</acceptance_criteria>
