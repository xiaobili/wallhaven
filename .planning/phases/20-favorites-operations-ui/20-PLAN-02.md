Add favorite button and collection indicator to wallpaper cards in WallpaperList.vue.

### Task 2.1: Add favorite button to wallpaper card template

<read_first>
- `src/components/WallpaperList.vue` — Current card structure with thumb-checkbox and thumb-btn-fav
- `src/components/favorites/CollectionDropdown.vue` — Dropdown component for collection selection
</read_first>

<action>
Modify `src/components/WallpaperList.vue`:

1. **Add props for favorites:**
```typescript
const props = defineProps<{
  pageData: TotalPageData;
  loading: boolean;
  error: boolean;
  selectedIds?: string[];
  favoriteIds?: Set<string>;  // NEW: Set of favorited wallpaper IDs
}>();
```

2. **Add emits for favorite operations:**
```typescript
const emit = defineEmits<{
  // ... existing emits
  'toggle-favorite': [item: WallpaperItem, event: MouseEvent];  // NEW
}>();
```

3. **Add favorite button in template (after thumb-checkbox, before thumb-btn-fav):**
```vue
<!-- Favorite button (left side, near checkbox) -->
<div
  class="thumb-favorite-btn"
  :class="{ 'is-favorite': isFavorite(liItem.id) }"
  :title="isFavorite(liItem.id) ? '已收藏' : '添加到收藏'"
  @click.stop="emit('toggle-favorite', liItem, $event)"
>
  <i :class="isFavorite(liItem.id) ? 'fas fa-heart' : 'far fa-heart'" />
</div>
```

4. **Add favorite indicator (shows when favorited):**
```vue
<!-- Favorite indicator badge -->
<div
  v-if="isFavorite(liItem.id)"
  class="favorite-indicator"
  :title="'已收藏'"
>
  <i class="fas fa-heart" />
</div>
```

5. **Add isFavorite helper:**
```typescript
const isFavorite = (id: string): boolean => {
  return props.favoriteIds?.has(id) || false
}
</action>
