# Plan 1: Create CollectionDropdown Component

## Objective

Create a reusable dropdown component for selecting collections when adding/removing/moving wallpapers.

## Tasks

### Task 1.1: Create CollectionDropdown.vue component

<read_first>
- `src/components/WallpaperList.vue` — Reference for dropdown positioning and styling patterns
- `src/composables/favorites/useFavorites.ts` — Interface for favorites operations
- `src/composables/favorites/useCollections.ts` — Interface for collections data
- `src/types/favorite.ts` — Collection and FavoriteItem types
</read_first>

<action>
Create `src/components/favorites/CollectionDropdown.vue` with:

1. **Props interface:**
```typescript
interface Props {
  wallpaperId: string
  wallpaperData: WallpaperItem
  visible: boolean
  position: { x: number; y: number }
}
```

2. **Template structure:**
```vue
<div v-if="visible" class="collection-dropdown" :style="dropdownStyle">
  <!-- Quick add to default collection -->
  <div class="dropdown-item quick-add" @click="quickAdd">
    <i class="fas fa-star" />
    <span>快速添加到"{{ defaultCollection?.name }}"</span>
  </div>
  <div class="dropdown-divider" />
  <!-- Collection list with checkboxes -->
  <div
    v-for="collection in collections"
    :key="collection.id"
    class="dropdown-item"
    :class="{ 'selected': isInCollection(collection.id) }"
    @click="toggleCollection(collection.id)"
    @contextmenu.prevent="showMoveMenu(collection.id)"
  >
    <i v-if="isInCollection(collection.id)" class="fas fa-check" />
    <i v-else class="far fa-square" />
    <span>{{ collection.name }}</span>
    <button
      v-if="isInCollection(collection.id) && !collection.isDefault"
      class="remove-btn"
      @click.stop="removeFromCollection(collection.id)"
    >
      <i class="fas fa-times" />
    </button>
  </div>
</div>
```

3. **Script logic:**
- Import `useFavorites` and `useCollections` from `@/composables`
- Load collections on mount via `useCollections().load()`
- Implement `isInCollection(collectionId)` using `useFavorites().favorites`
- Implement `quickAdd()` to add to default collection
- Implement `toggleCollection(collectionId)` to add/remove
- Implement `removeFromCollection(collectionId)` using `useFavorites().remove()`
- Implement `showMoveMenu(collectionId)` for move operation (emit event)
- Position dropdown using `position` prop with `fixed` positioning

4. **Styles:**
- Width: 180px
- Background: #2a2a2a
- Border-radius: 4px
- Box-shadow: 0 4px 12px rgba(0,0,0,0.3)
- Z-index: 1000
- Item hover: background #3a3a3a
- Selected items: checkmark visible
- Remove button: appears on hover for selected items
</action>
