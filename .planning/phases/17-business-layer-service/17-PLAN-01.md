## Objective

Create `collections.service.ts` that provides business logic for collection management with memory caching and error handling.

<acceptance_criteria>
1. File exists at src/services/collections.service.ts
2. Class named CollectionsServiceImpl with private cache fields
3. getAll() returns IpcResponse<Collection[]> with cache optimization
4. getById(id) returns IpcResponse<Collection | null>
5. getDefault() returns IpcResponse<Collection | null> for isDefault=true collection
6. create(name) returns IpcResponse<Collection> and invalidates cache
7. rename(id, name) returns IpcResponse<Collection> and invalidates cache
8. delete(id) returns IpcResponse<void> and invalidates cache
9. clearCache() sets cachedCollections and cachedData to null
10. Export: export const collectionsService = new CollectionsServiceImpl()
</acceptance_criteria>

<code_reference>
```typescript
// Expected structure
class CollectionsServiceImpl {
  private cachedCollections: Collection[] | null = null
  private cachedData: FavoritesData | null = null

  async getAll(): Promise<IpcResponse<Collection[]>>
  async getById(id: string): Promise<IpcResponse<Collection | null>>
  async getDefault(): Promise<IpcResponse<Collection | null>>
  async create(name: string): Promise<IpcResponse<Collection>>
  async rename(id: string, name: string): Promise<IpcResponse<Collection>>
  async delete(id: string): Promise<IpcResponse<void>>
  clearCache(): void
}

export const collectionsService = new CollectionsServiceImpl()
```
