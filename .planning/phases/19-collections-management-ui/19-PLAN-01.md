# PLAN 01: Add Favorites Route

## Objective

Add `/favorites` route to the Vue Router configuration to enable navigation to the favorites page.

## Tasks

### Task 1: Add Favorites Route Entry

<read_first>
- src/router/index.ts
</read_first>

<action>
1. Open `src/router/index.ts`
2. Add a new route entry in the `routes` array, positioned after the `/download` route and before `/setting`:

```typescript
{
  path: '/favorites',
  name: 'FavoritesPage',
  component: () => import('@/views/FavoritesPage.vue'),
  meta: {
    title: '我的收藏',
    icon: 'fas fa-heart',
  },
},
```

3. The route should be inserted at line 37 (after the download route closing brace)
</action>

<acceptance_criteria>
- `grep -n "path: '/favorites'" src/router/index.ts` returns a match

- `grep -n "我的收藏" src/router/index.ts` returns a match
</acceptance_criteria>

## Verification

1. Router config is valid TypeScript
2. Route follows existing pattern (path, name, component, meta)
3. Component import path is correct (`@/views/FavoritesPage.vue`)
