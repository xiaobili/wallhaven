# Phase 30, Plan 01: Create Splash Window Infrastructure

Create the splash window foundation in Electron main process.

## Objective

Add a dedicated splash window that appears before the main window, implementing all Phase 30 requirements: frameless design, centered positioning, and dark theme matching.

## Tasks

<task>
<id>01</id>
<name>创建 splash.html 文件</name>
<objective>Create the splash HTML file with inline CSS</objective>

<read_first>
- electron/main/index.ts (existing window creation pattern)
</read_first>

<action>
Create file: electron/renderer/splash.html

Content:
- Basic HTML5 structure
- Inline CSS with background: #1a1a1a (D-05 dark theme)
- Empty body (logo animation will be added in Phase 31)
- No external CSS references, no JS (fastest loading)
- CSS resets: margin: 0, padding: 0, overflow: hidden

Use center: true in BrowserWindow options handles centering.
</action>

<acceptance_criteria>
- File exists at electron/renderer/splash.html ✓
- HTML has valid HTML5 doctype and structure ✓
- Background color is #1a1a1a (grep "background.*#1a1a1a" ✓
- No external CSS or JS references (grep -E "<link|<script" returns nothing) ✓
- Body has margin: 0, padding: 0, overflow: hidden ✓
</acceptance_criteria>
</task>

<task>
<id>02</id>
<name>在 index.ts 中添加 splash 窗口创建逻辑</name>
<objective>Add splash window creation in electron/main/index.ts</objective>

<read_first>
- electron/main/index.ts (full file, current createWindow implementation)
</read_first>

<action>
Modify electron/main/index.ts:

1. Add `let splashWindow: BrowserWindow | null = null` at top scope (D-01: inline in index.ts)

2. In app.whenReady() callback, BEFORE createWindow() call (D-04):
   - Create splashWindow = new BrowserWindow({
       width: 400, height: 300, (D-03)
       frame: false, (D-02: frameless)
       resizable: false,
       center: true, (D-03: centered)
       show: false,
       backgroundColor: '#1a1a1a', (D-05)
       webPreferences: { sandbox: false }
     })

2. Load splash.html:
   - dev AND prod: splashWindow.loadFile(path.join(__dirname, '../renderer/splash.html')) (D-02)

3. On ready-to-show: splashWindow.show()

4. Keep mainWindow.show: false as-is (Phase 32 handles timing)
</action>

<acceptance_criteria>
- splashWindow variable declared at top scope as BrowserWindow | null ✓
- splashWindow created in app.whenReady() BEFORE createWindow() call ✓
- frame: false set on splashWindow (grep "frame:.*false") ✓
- center: true set on splashWindow ✓
- width: 400, height: 300 ✓
- backgroundColor: '#1a1a1a' set ✓
- resizable: false set ✓
- loadFile() used with correct path to splash.html ✓
- ready-to-show event shows splashWindow ✓
- mainWindow.show() call remains commented out / show: false ✓
</acceptance_criteria>
</task>

## Verification Criteria

Run `npm run dev` and verify:
1. Splash window appears before main window (SPLASH-01)
2. Splash window has no title bar or window controls (SPLASH-02)
3. Splash window is centered on screen (SPLASH-03)
4. Splash window background is dark #1a1a1a (SPLASH-04)
5. No errors in console related to splash window
6. Main window does not show (stays hidden - Phase 32 handles timing)

## Must Haves

<must_haves>
- Splash window is created before main window
- Splash window is frameless (no title bar, no controls)
- Splash window is centered on screen
- Splash window background color is #1a1a1a
- All code is inline in electron/main/index.ts
- Splash window loads local HTML file
</must_haves>
