# Milestone v2.5: 壁纸收藏功能

**Status:** ✅ SHIPPED 2026-04-29
**Phases:** 16-22
**Total Plans:** 23

## Overview

为壁纸浏览器添加本地收藏功能，支持自定义收藏夹分类管理。用户可以创建、重命名、删除收藏夹，将喜欢的壁纸添加到收藏夹，并支持快速收藏到默认收藏夹。

## Milestone Summary

**Key Decisions:**
- Decision: O(1) favoriteIds Set for fast lookups (Rationale: Performance for large collections)
- Decision: Left-click quick favorite, right-click selector (Rationale: Efficient UX for common case)
- Decision: Wallpaper can belong to multiple collections (Rationale: Flexible organization)
- Decision: KeepAlive caching for FavoritesPage (Rationale: Instant tab switching)
