# 🚀 快速启动指南

## 第一步：安装依赖

```bash
# 安装基础依赖
npm install

# 安装wallpaper包（用于设置桌面壁纸）
npm install wallpaper@latest --legacy-peer-deps
```

## 第二步：启动应用

```bash
npm run dev
```

这将启动Electron开发环境，应用窗口会自动打开。

## 第三步：测试功能

### 1️⃣ 配置下载目录（首次使用必须）

1. 点击左侧菜单的"设置"图标
2. 在"下载设置"区域，点击"浏览"按钮
3. 选择一个文件夹作为壁纸下载目录
4. 点击"保存设置"

### 2️⃣ 浏览和下载在线壁纸

1. 点击"在线壁纸"菜单
2. 使用搜索栏筛选壁纸（可选）
3. 滚动页面加载更多壁纸
4. 点击任意壁纸卡片上的：
   - **下载图标** 💾 - 下载到本地
   - **桌面图标** 🖥️ - 直接设置为桌面壁纸

### 3️⃣ 查看本地壁纸

1. 点击"本地列表"菜单
2. 自动显示下载目录中的所有图片
3. 可以执行以下操作：
   - 点击图片预览大图
   - 点击左上角图标设为壁纸
   - 点击右上角图标删除文件
   - 点击"打开文件夹"按钮查看原文件

### 4️⃣ 管理下载任务

1. 点击"下载中心"菜单
2. 查看正在下载和已完成的任务
3. 可以暂停、恢复、取消下载
4. 点击文件夹图标在系统中打开文件位置

## 🐛 常见问题

### Q: 启动后显示"网络异常"？
**A**: 
1. 检查网络连接
2. 确认已重启开发服务器（修改配置后必须重启）
3. 访问 `/api-test` 页面测试API连接

### Q: 设置壁纸失败？
**A**: 
- **macOS**: 需要在"系统偏好设置 > 安全性与隐私 > 隐私"中授予屏幕录制权限
- **Windows**: 确保有管理员权限
- **Linux**: 需要安装相应的桌面环境工具

### Q: 下载目录无法选择？
**A**: 
确保在Electron窗口中操作，而不是浏览器。只有在Electron环境中才能调用系统对话框。

### Q: 本地列表显示为空？
**A**: 
1. 确认已设置下载目录
2. 确认目录中有图片文件（jpg, png, gif, bmp, webp）
3. 点击"刷新"按钮重新加载

### Q: wallpaper模块找不到？
**A**: 
运行以下命令安装：
```bash
npm install wallpaper@latest --legacy-peer-deps
```
